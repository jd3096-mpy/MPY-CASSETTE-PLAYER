## 按键功能简介

*按住STOP键，再按POWER键开机则进入boot下载模式，可直接拖拽UF2下载固件*

|      | STOP     | PLAY          | N/V+     | N/V-     | POWER    |
| ---- | -------- | ------------- | -------- | -------- | -------- |
| 短按 | 暂停     | 播放          | 有声快进 | 无声快退 | 屏幕开关 |
| 中按 |          | 切换封面/歌名 | 音量加   | 音量减   | 关机     |
| 长按 | 歌曲选择 | 设置菜单      | 下一首   | 上一首   | 硬关机   |
